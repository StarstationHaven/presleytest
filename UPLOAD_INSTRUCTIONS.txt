================================================================================
STEP 2: UPLOAD FILES TO GITHUB
================================================================================

After creating the repository, you'll see an empty repository page.

Option A - Drag and Drop (Easiest):
1. Click "Add file" → "Upload files"
2. Drag and drop ALL files from this zip into the upload area
3. Add commit message: "Initial commit: Presley EXCEL website"
4. Click "Commit changes"

Option B - Upload Folder by Folder:
1. Extract this zip file
2. Go to your GitHub repository
3. Click "Add file" → "Upload files"
4. Upload each folder/file:
   - First upload: dist/ folder (MOST IMPORTANT)
   - Then upload: src/ folder
   - Then upload: public/ folder
   - Then upload: all root files (index.html, package.json, etc.)

================================================================================
STEP 3: CONFIGURE GITHUB PAGES
================================================================================

1. Go to: https://github.com/StarstationHaven/presley-excel-website
2. Click "Settings" (top right)
3. Scroll to "Pages" section (left sidebar)
4. Under "Build and deployment":
   - Source: Select "Deploy from a branch"
   - Branch: Select "main" (or "master")
   - Folder: Select "/" (root)
5. Click "Save"
6. Wait 1-2 minutes for deployment

================================================================================
STEP 4: VIEW YOUR LIVE WEBSITE
================================================================================

Your website will be live at:
https://StarstationHaven.github.io/presley-excel-website/

Go to Settings → Pages to confirm the deployment status.

================================================================================
TROUBLESHOOTING
================================================================================

Issue: Website shows 404 error
Solution: Make sure you uploaded the dist/ folder to the root of the repository

Issue: Styles or images not loading
Solution: 
- Clear browser cache (Ctrl+Shift+Delete)
- Hard refresh (Ctrl+Shift+R or Cmd+Shift+R)
- Wait 2-3 minutes for GitHub to deploy

Issue: Can't find upload button
Solution: 
- Make sure you're in the repository (not your profile)
- Look for the green "Code" button
- Click the dropdown next to it
- Select "Upload files"

Issue: GitHub Pages not showing deployment
Solution:
- Go to Settings → Pages
- Check that source is "Deploy from a branch"
- Check that branch is "main" or "master"
- Wait 2-3 minutes and refresh
